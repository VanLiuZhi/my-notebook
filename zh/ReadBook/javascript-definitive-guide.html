关于分号

关于分号，分号可以不要，如果你不要分号，语句会被解释器连起来。

比如：

  var x = y + z
  (a+b).result()

实际是 var x = y + z(a+b).result() 分号是语句的结束，你不加分号解释器会自己来处理，它处理不了的语句会自己加分号，等等一些规则，所以养成加分号是好习惯，避免前面的语句改了，没有分号语句被解释器组合了。
